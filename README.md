# BE

This store is based on [spawanie.com](https://spawanie.com/).

How to install:
1. clone repo, go to /BE/PrestaShop-8.1.x
4. `sudo chmod 777 -R .`
6. `sudo docker-compose up`
7. ctrl + c
8. `sudo docker-compose down`
9. `sudo chmod 777 -R .`
10. `git reset --hard`
11. `docker-compose up`
